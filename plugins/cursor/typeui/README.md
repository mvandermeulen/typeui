# TypeUI Cursor Plugin

TypeUI connects Cursor to the TypeUI MCP server so it can use curated design systems, UI prompts, and layout variations while building interfaces.

## Install

After TypeUI is available on the Cursor Marketplace, open Cursor Agent chat and run:

```text
/add-plugin typeui
```

You can also search for TypeUI in the Cursor Marketplace.

After installation, Cursor will connect to TypeUI MCP. Sign in with TypeUI if Cursor asks you to authorize the connection.

## MCP server

This plugin registers the TypeUI remote MCP server:

```text
https://mcp.typeui.sh
```
